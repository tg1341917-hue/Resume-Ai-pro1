# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, Machine Learning, and Google Gemini AI.

This project analyzes a resume, calculates ATS compatibility, compares skills with the target career, provides career recommendations, improves resumes using AI, and generates professional resume PDFs.

---

## ✨ Features

- 📄 Upload Resume (PDF)
- 🎯 ATS Score Analysis
- 🧠 AI Career Report
- 📊 Career Readiness Dashboard
- 🔍 Skill Extraction
- 📈 Career Match Analysis
- 📚 Missing Skills Detection
- 🤖 AI Resume Improvement
- 📝 Resume Builder
- 📥 Download Professional Resume PDF
- 🌐 Streamlit Web Application

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Resume-Ai-pro1.git
```

Go inside the folder

```bash
cd Resume-Ai-pro1
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```

---

## 🔑 Environment Variable

Create

```
.streamlit/secrets.toml
```

Add

```toml
GEMINI_API_KEY="YOUR_API_KEY"
```
