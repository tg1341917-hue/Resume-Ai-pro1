# 📌 Overview

ResumeAI Pro is an intelligent web application that helps students and professionals analyze resumes, calculate ATS scores, identify missing skills, receive AI-powered career guidance, improve resumes, and generate professional ATS-friendly resumes.

The project combines Machine Learning, Natural Language Processing, PDF Processing, and Generative AI to provide a complete resume analysis experience.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/tg1341917-hue/Resume-Ai-pro1.git
```

---

## Go to Project Folder

```bash
cd Resume-Ai-pro1
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create Secrets File

Create

```text
.streamlit/secrets.toml
```

Add

```toml
GEMINI_API_KEY="YOUR_API_KEY"
```

---

## Run Project

```bash
streamlit run app.py
```

# 📊 Modules

### Resume Parser

Extracts text from uploaded PDF resumes.

---

### Skill Detection Engine

Detects technical and professional skills using keyword matching.

---

### ATS Engine

Calculates:

- ATS Score
- Resume Quality
- Education Score
- Experience Score
- Career Readiness

---

### Career Recommendation Engine

Compares resume skills with required career skills.

---

### AI Engine

Uses Google Gemini AI for:

- Resume Analysis
- Resume Improvement
- Resume Generation
- Career Guidance

---

### PDF Engine

Creates professional ATS-friendly resumes in PDF format.
